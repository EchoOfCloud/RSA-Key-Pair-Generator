# RSA密钥对生成工具

## 项目简介

RSA密钥对生成工具是一个基于Web技术开发的静态网页应用，用于根据RSA算法的核心参数（素数p、素数q和公钥指数e）生成标准格式的PEM私钥文件。该工具采用纯JavaScript实现，所有计算均在浏览器端完成，确保了数据安全性和使用便捷性。

## 技术支持

本项目由阿里云ESA提供加速、计算和保护

![阿里云ESA](https://img.alicdn.com/imgextra/i3/O1CN01H1UU3i1Cti9lYtFrs_!!6000000000139-2-tps-7534-844.png)

## 功能特性

- ✅ **RSA私钥生成**：根据输入的p、q、e参数，自动计算生成完整的RSA私钥
- ✅ **PEM格式输出**：生成符合标准的PEM格式私钥，可直接用于各种RSA加密解密场景
- ✅ **中间值展示**：实时显示密钥生成过程中的关键中间值
- ✅ **示例数据测试**：提供内置示例数据，方便用户快速测试工具功能
- ✅ **私钥下载**：支持将生成的PEM私钥直接下载为文件
- ✅ **纯前端实现**：无任何外部依赖，直接在浏览器中运行
- ✅ **响应式设计**：适配不同屏幕尺寸，简洁易用

## 中间值说明

生成过程中会显示以下中间计算值：

| 参数 | 说明 |
|------|------|
| e | 公钥指数 |
| d | 私钥指数 |
| n | 模数 (p*q) |
| dmp1 | d mod (p-1) |
| dmq1 | d mod (q-1) |
| iqmp | q^-1 mod p |

## 技术栈

- **前端框架**：纯HTML5 + CSS3 + JavaScript（ES6+）
- **算法实现**：
  - 扩展欧几里得算法用于计算模逆元
  - 完整的ASN.1 DER编码，确保生成的PEM格式符合标准
  - 支持BigInt运算，处理大素数计算
- **无外部依赖**：无需安装任何库或框架

## 快速开始

### 方法一：直接打开HTML文件

1. 下载项目文件
2. 在浏览器中直接打开 `rsa_pem_generator.html` 文件
3. 开始使用工具

### 方法二：本地HTTP服务器

1. 克隆或下载项目到本地
2. 在项目目录中运行HTTP服务器：
   ```bash
   # 使用Python 3
   python -m http.server 8000
   
   # 使用Node.js
   npx serve
   ```
3. 在浏览器中访问 `http://localhost:8000/rsa_pem_generator.html`

## 使用说明

1. **输入参数**：在对应输入框中填写素数p、素数q和公钥指数e（通常为65537）
2. **生成私钥**：点击「生成PEM私钥」按钮
3. **查看结果**：
   - 下方显示生成的PEM格式私钥
   - 中间值区域显示各计算步骤的结果
4. **下载私钥**：点击「下载PEM文件」按钮，将私钥保存为文件


## 部署说明

### 静态网站托管

该项目是纯静态HTML页面，可以部署到任何静态网站托管服务，如：

- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages

### 使用npm部署

项目已包含 `package.json` 文件，支持通过npm进行部署：

1. 确保已安装Node.js和npm
2. 在项目目录中运行：
   ```bash
   npm run build
   ```
3. 将生成的文件部署到目标服务器

1. **RSA算法学习**：直观展示RSA密钥生成过程，帮助理解RSA算法原理
2. **密钥生成工具**：快速生成RSA私钥，用于开发测试或小型应用

## 注意事项

- 本工具仅用于学习和测试目的，请勿在生产环境中使用弱素数
- 生成的私钥请妥善保管，避免泄露
- 浏览器端计算大素数可能会消耗较多资源，建议使用现代浏览器
